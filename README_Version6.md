# SDG6-Explorers

Interactive single-page site focused on SDG 6 (Clean Water & Sanitation).
Features:
- Four SDG6 subtopics (Definition, Causes, Effects, Solutions) with interactive tools.
- Water Quality Checker and Water Impact Estimator.
- Feedback form that posts to Formspree and stores local backups (localStorage).
- Export saved feedback to CSV.
- Designed for GitHub Pages hosting.

To run locally:
1. Save files to a folder.
2. Open index.html in any modern browser.

To deploy to GitHub Pages, push to the repository `YongZe12/SDG6-Explorers` and enable Pages on the `main` branch.
